Denna uppgift gick ut på att fortsätta bekanta sig med Github genom att regelbundet ladda upp arbetet med commits och meddelanden.

Automatisering & publicering

Skapa en grundstruktur med HTML, CSS, JavaScript och bilder för webbutveckling med hjälp av NodeJs för att automatisera uppgifterna.

De uppgifter som skall automatiskeras är:
  • Minifiering av CSS
  • Transpilering och minifiering av JavaScript
  • Optimering av bilder
  • Bakåtkompabilitet i genererade filer
  • Kopiering av arbetsfiler

Skapa Demowebbplats

Skapa en demowebbplats som innehåller HTML, CSS och JavaScript som består av:
  • Startsida
    • En startsida som innehåller info om uppgiften
  • Bildsida
    • En undersida som innehåller minst 5 foton
  • Arbetsprocess sida
    • Undersida som beskriver i detalj hur arbetsprocessen för utveckling och publicering fungerar.
      Det ska även innehålla information om vad som är automatiserat och vilka verktyg som har använts för detta.

Webbsidan ska sen publiceras till en webbhost som Github där den också ska publiceras automatiskt vid uppdatering.
